Recording

The Recording feature of the pylon Viewer allows you to record video clips or sequences of still images.

The Recording feature is available in the Windows version of the pylon Viewer only.

The video clips or still images recorded using this feature will be displayed in the image display area of the pylon Viewer. The resulting videos can be played back with any standard media player already installed on your computer, e.g., Windows Media Player.

There are several easy-to-configure options in the Options dialog to customize the recording feature for your needs. To open this dialog, go to Tools > Options, and then click Recording.

There are two pages with recording options:

  • the Recording page
  • the Sampling page

Recording Page

On the Recording page, you have the following options:

  • Output folder: Determines where the resulting video clips and images are going to be stored. Use the ... button to navigate to your preferred location.
  • Video (AVI): Check this option to record a video clip. You can record video clips of up to 2 GB in size. Once that threshold is reached, the recording will stop and an error message will be displayed.
    • Set fixed playback speed: Determines at what speed (in frames per second, fps) a recorded video clip will be played back. If you don't check this box, the clip will be played back in real-time, i.e., at the same speed that it has been recorded.

The minimum playback speed is 1 frame per second. This means that if you recorded a video at the speed of 1 frame per minute, it would still be played back at 1 frame per second.

    • Use compression: If you check this option and click the ... button, the Windows Compression Options dialog opens where you can choose the desired codec.
  • Sequence of still images: Check this option to record a series of still images.
    • Output format: Determines the file format of the resulting images. You can choose between bitmap and tiff files.
  • Recording buffer size: Determines how many images can be stored in the buffer during recording.
    The image buffer acts like a cache to store images if they can't be read out of the camera quickly enough. You can set the desired number of images to be stored in the buffer and during recording the status bar in the image display area will show you how much the buffer is being used (in percent).  
  • Show options every time a recording is started: Uncheck this box if you don't want to see the recording options every time you start a recording.

Sampling Page

On the Sampling page, you have the following options:

  • Frame-based recording: Determines the rate at which images are recorded based on frames.
    • Record a frame every x frame(s): Lets you select to capture every frame, every third frame, every fifth frame etc.
    • Stop recording after x recorded frames: Check this option to stop a recording after a certain number of frames have been recorded.
  • Time-based recording: Determines the rate at which images are recorded based on time.
    • Record a frame every ...: Lets you select the time interval for recording frames.
    • Stop recording after ...: Check this option to stop a recording after a certain period of time.

Image Display Area During Recording

When you're recording a video or a sequence of still images, a red recording icon will appear in the image display area. You can pause the recording by clicking the  button in the toolbar, by pressing the F10 button or by clicking Camera in the menu bar and selecting Pause. The recording icon will then change to a pause icon. To restart the recording, click the button again.

At the bottom of the image display area is a status bar that shows you the status of the camera. It also shows the number of recorded and dropped frames as well as the image buffer usage. Frames are dropped when the image buffer has reached its maximum capacity. To remedy that situation, increase the buffer size on the Sampling page of the Options dialog as required.
